# История изменений

## 0.2.0 — 2026-07-23

### Добавлено

- Мультивалютность: канал `channel-rub` (RUB) + `default-channel` (USD)
- Цены переключаются по локали: рубли для `ru`, доллары для `en`
- 30 новых товаров (по 5 в processors, videoCards, motherboards, ram, storage, powerSupplies)
- Реальные цены в USD для всех ~100 товаров

### Изменено

- Каталог теперь исключает сборки из сортировки/общего списка
- Сортировка читается из URL (`?sort=price-desc`) при загрузке
- URL синхронизируется с категорией и сортировкой
- `formatPrice` зависит от локали: `$1,200` для en, `10 700 ₽` для ru
- Все API-роуты принимают `locale` для выбора канала

### Исправлено

- Старые замыкания locale в `catalog-client`, `popular-products`, `builds-client`, `compare-client`
- Хардкод русских строк в compare (теперь через `t()`)
- Название категории "Корпуса" → "Корпус"

### Удалено

- Мёртвый код: `data/products.ts`, `data/categories.ts`, `data/builds.ts`, `data/gql-client.ts`, `use-catalog.ts`

## 0.1.0 — 2026-07-22

### Добавлено

- Страница сравнения с выбором категории, подбором двух товаров через оверлей и сравнением характеристик
- Цветовая индикация результатов: зелёный (лучше), красный (хуже), золотой (нейтрально)
- Числовое сравнение с учётом направления (больше/меньше — лучше)
- API-ручки: `/api/category-products`, `/api/products`, `/api/popular`, `/api/builds`
- Контекст корзины и кнопка добавления в корзину
- Страница готовых сборок со списком комплектующих
- Локализация (ru/en) через next-intl
- Каталог с бесконечной прокруткой, фильтрацией по категориям и сортировкой по цене
- Хиро-секция с анимированными точками и сеткой категорий на главной
