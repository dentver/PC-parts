# PC Parts

## URL для демонстрации
https://pc-parts-iota.vercel.app/ru

## Стек

- **Фреймворк** — Next.js 16 (App Router)
- **Язык** — TypeScript
- **Стили** — SCSS Modules
- **Локализация** — next-intl (ru, en)
- **Коммерция** — Saleor GraphQL API
- **Шрифты** — Unbounded (заголовки), Rubik (текст), JetBrains Mono (моноширинный)

## Запуск

```bash
npm install
npm run dev
```

## Структура проекта

```
src/
  app/
    [locale]/          # Локализованные маршруты (ru, en)
      catalog/         # Каталог с фильтром по категориям
      builds/          # Готовые сборки
      compare/         # Сравнение комплектующих
      cart/            # Корзина
    api/
      category-products/  # Товары по категории
      products/           # Все товары (пагинация, сортировка)
      popular/            # Популярные товары
      builds/             # Готовые сборки
  components/          # Общие UI-компоненты
  data/
    types.ts           # Основные типы (Product, Spec, Build и т.д.)
    api.ts             # Saleor GraphQL клиент
  context/
    cart-context.tsx   # Состояние корзины
```

## Возможности

- **Каталог** — просмотр товаров по категориям или всех сразу, бесконечная прокрутка, сортировка по цене
- **Сборки** — готовые конфигурации ПК с полным списком комплектующих
- **Сравнение** — поочередное сравнение двух комплектующих в одной категории
- **Корзина** — добавление и удаление товаров

## Страница сравнения

Выберите категорию, затем два товара из оверлея. Таблица сравнивает все характеристики:

- **Зелёный** — лучшее значение (числовые характеристики)
- **Красный** — худшее значение
- **Золотой** — равные, нечисловые или несравнимые характеристики

Числовые характеристики учитывают направление: для частот, ядер, памяти и т.д. больше — лучше; для TDP, задержек, напряжения и цены — меньше — лучше.

## API

Все ручки возвращают JSON. Ошибки: `{ "error": "message" }` со статусом 500 (или 400 при отсутствии обязательных параметров).

### `GET /api/category-products`

Товары выбранной категории.

| Параметр    | Тип    | По умолчанию | Описание                |
| ----------- | ------ | ------------ | ----------------------- |
| `category`  | string | —            | Slug категории (обяз.)  |
| `first`     | number | 10           | Размер страницы (1–50)  |
| `after`     | string | null         | Курсор пагинации        |

### `GET /api/products`

Все товары с фильтрацией и сортировкой.

| Параметр    | Тип    | По умолчанию | Описание                     |
| ----------- | ------ | ------------ | ----------------------------- |
| `category`  | string | null         | Фильтр по slug категории      |
| `first`     | number | 10           | Размер страницы (1–50)        |
| `after`     | string | null         | Курсор пагинации              |
| `sort`      | string | null         | `price-asc` / `price-desc`    |

### `GET /api/popular`

Возвращает `{ "products": [...] }` — до 6 популярных товаров (при отсутствии заказов — по одному из каждой категории).

### `GET /api/builds`

Возвращает массив готовых сборок с комплектующими, полной ценой и метаданными.
